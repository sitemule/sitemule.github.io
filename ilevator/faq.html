#What does Porcelain API mean?

Porcelain API is a more user-friendly API which handles all the more low-level API of ILEvator.

“Porcelain” is the material from which toilets are usually made (and sometimes other fixtures such as washbasins). This is distinct from “plumbing” (the actual pipes and drains), where the porcelain provides a more user-friendly interface to the plumbing. — StackOverflow
